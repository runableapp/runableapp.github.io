Simple, Local, Self-Contained

Software should be simple, local, and self-contained. It should be a single executable that you can download and run. You shouldn’t need an installer or any other software prerequisites. At the very least, dependencies should be minimized.
If an installer is required, it should not get in its own way. This involves installing the minimum necessary and ensuring that 'uninstall' actually undoes the install. You should be able to remove the software and have your system exactly as you found it, with no traces of the software installation behind.
